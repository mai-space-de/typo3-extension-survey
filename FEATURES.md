## Survey Records

* Survey records — surveys with title, description, and configurable question sets
* Question types — single choice, multiple choice, text, scale, and date input
* Multi-step wizard — paginated survey flow with progress indicator

## Submissions

* Anonymous submissions — surveys can be completed without a frontend account
* Authenticated submissions — optionally restrict to logged-in users via `mai_account`
* Duplicate prevention — optional cookie or session-based deduplication
* Results backend module — view, filter, and export submissions as CSV

## Mitmach-Matrix (Volunteer Matching)

* Interactive grid survey — volunteers select time availability, language skills, and interests
* Dynamic questionnaire — multi-step survey flow generated based on selections
* Admin notification — submission notification dispatched via `mai_mail`
